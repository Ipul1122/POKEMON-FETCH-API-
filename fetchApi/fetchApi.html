FETCH API

fetch = fucntion yang digunakan untuk membuat permintaan HTTP untuk mengambil sumber daya.
(JSON style data, gambar, file) Menyederhanakan pengambilan data asinkron dalam javascript dan digunakan untuk berinteraksi dengan API untuk mengambil dan mengirim data secara asinkron melalui web
fetch(url,{method : "GET", "POST", "DELETE"})
